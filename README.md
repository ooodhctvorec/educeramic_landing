# Статичный сайт «Студии керамики для школ и детских садов»

Это обычный HTML/CSS/JS-сайт, не привязанный к Tilda.

## Страницы

- `index.html` — главная
- `detskiy-sad.html` — страница детского сада
- `shkola.html` — страница школы
- `tech.html` — технические требования

## Фото и логотипы

Фотографии и логотипы нужно скачать из архива:
https://disk.yandex.ru/d/aRTiLwVQHPqh8w

После скачивания положите папки с фотографиями рядом с HTML-файлами, чтобы структура была такой:

```text
index.html
detskiy-sad.html
shkola.html
tech.html
assets/
Логотипы/
Сады/
Сад/
Школы/
```

В HTML прописаны относительные пути из ТЗ, включая кириллицу, пробелы и спорные места исходного документа:

- `Сады/Сад300 Фото1.jpg`
- `Сад/Сад300 Фото2.jpg`
- `Школы/Школа15000 Фото27.jpg`
- `Школы/Школа1500  Фото28.jpg` — с двумя пробелами перед `Фото28.jpg`

Если фактические имена файлов в архиве отличаются, переименуйте файл или поправьте `src` в HTML.

## Формы

Формы сейчас являются HTML-заглушками. Для реального сайта подключите:

- серверный обработчик;
- amoCRM/Bitrix24;
- Telegram-бот;
- Google Forms;
- любой другой сервис приема заявок.

## Запуск локально

Просто откройте `index.html` в браузере. Для проверки путей лучше запустить локальный сервер:

```bash
python3 -m http.server 8080
```

Затем открыть:

```text
http://localhost:8080
```
